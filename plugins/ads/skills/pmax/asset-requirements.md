# PMax Asset Requirements

Official Google Ads specifications for Performance Max creative assets.

---

## Text Assets

### Headlines

| Specification | Requirement |
|--------------|-------------|
| **Minimum** | 3 |
| **Maximum** | 15 |
| **Character limit** | 30 per headline |
| **Required variations** | At least 1 with brand name |

**Best Practices:**
- Use all 15 slots for maximum combinations
- Mix benefit-focused and feature-focused
- Include numbers and statistics when relevant
- Vary length (some short, some using full 30 chars)
- Include call-to-action in at least one

**Examples:**
```
Fast Shipping on All Orders
Save 20% Today Only
Award-Winning Service
[Brand Name] - Since 2010
Free 30-Day Returns
Top Rated by Customers
```

### Long Headlines

| Specification | Requirement |
|--------------|-------------|
| **Minimum** | 1 |
| **Maximum** | 5 |
| **Character limit** | 90 per long headline |

**Best Practices:**
- Expand on short headlines with more detail
- Include value proposition clearly
- Can include CTA + benefit combined
- Use for YouTube and Discovery placements

**Examples:**
```
Get Premium Quality [Product] Delivered to Your Door - Free Shipping Over $50
Join 10,000+ Happy Customers Who Switched to [Brand] This Year
```

### Descriptions

| Specification | Requirement |
|--------------|-------------|
| **Minimum** | 2 |
| **Maximum** | 5 |
| **Character limit** | 90 per description |

**Best Practices:**
- First description is most important
- Include key differentiators
- Add social proof where possible
- End with clear CTA

**Examples:**
```
Shop our award-winning collection with free shipping and easy returns. Order today!
Trusted by 50,000 customers. Premium quality at affordable prices. Shop now.
```

### Business Name

| Specification | Requirement |
|--------------|-------------|
| **Character limit** | 25 |
| **Required** | Yes |

### Call to Action

Select from Google's predefined options or use "Automated":
- Learn More
- Get Quote
- Apply Now
- Sign Up
- Contact Us
- Subscribe
- Download
- Book Now
- Shop Now
- Buy Now
- Order Now
- Get Started
- Automated (recommended)

## Image Assets

### Landscape Image (Required)

| Specification | Requirement |
|--------------|-------------|
| **Aspect ratio** | 1.91:1 |
| **Recommended size** | 1200 x 628 px |
| **Minimum size** | 600 x 314 px |
| **Minimum required** | 1 |
| **Maximum allowed** | 20 |
| **File size** | Max 5 MB |
| **File types** | JPG, PNG |

### Square Image (Required)

| Specification | Requirement |
|--------------|-------------|
| **Aspect ratio** | 1:1 |
| **Recommended size** | 1200 x 1200 px |
| **Minimum size** | 300 x 300 px |
| **Minimum required** | 1 |
| **Maximum allowed** | 20 |
| **File size** | Max 5 MB |
| **File types** | JPG, PNG |

### Portrait Image (Optional)

| Specification | Requirement |
|--------------|-------------|
| **Aspect ratio** | 4:5 |
| **Recommended size** | 960 x 1200 px |
| **Minimum size** | 480 x 600 px |
| **Minimum required** | 0 |
| **Maximum allowed** | 20 |
| **File size** | Max 5 MB |
| **File types** | JPG, PNG |

### Image Best Practices

**Do:**
- Use high-quality, professional images
- Show product in use/context
- Include people when appropriate
- Use bright, clear lighting
- Test multiple image styles

**Don't:**
- Use text overlays (Google adds text dynamically)
- Use collages or image grids
- Use blurry or pixelated images
- Use misleading imagery
- Use images with excessive white space

### Landscape Logo (Optional)

| Specification | Requirement |
|--------------|-------------|
| **Aspect ratio** | 4:1 |
| **Recommended size** | 1200 x 300 px |
| **Minimum size** | 512 x 128 px |
| **Minimum required** | 0 |
| **Maximum allowed** | 5 |
| **File size** | Max 5 MB |
| **File types** | PNG (with transparency preferred), JPG |

### Logo Best Practices

- Use transparent background (PNG)
- Ensure logo is legible at small sizes
- Center the logo in the image
- Don't add extra padding/margins
- Use vector-based source when possible

## Video Assets

### Specifications

| Specification | Requirement |
|--------------|-------------|
| **Minimum duration** | 10 seconds |
| **Recommended duration** | 15-30 seconds |
| **Hosting** | YouTube only |
| **Aspect ratios** | 16:9 (landscape), 1:1 (square), 9:16 (vertical) |
| **Minimum required** | 0 (but strongly recommended) |
| **Maximum allowed** | 5 |

### Video Best Practices

**Do:**
- Hook viewers in first 3 seconds
- Include branding early
- Optimize for sound-off viewing (captions)
- Include clear CTA
- Test multiple video lengths

**Don't:**
- Use videos under 10 seconds
- Rely solely on audio for message
- Use copyrighted music
- Include phone numbers in video

### Auto-Generated Videos

If you don't provide videos:
- Google creates videos from your images
- Quality is typically lower
- Less control over messaging
- Still shows on YouTube/video placements

**Recommendation:** Provide at least one custom video

## Asset Performance Ratings

Google rates each asset:

| Rating | Meaning | Action |
|--------|---------|--------|
| **Best** | Top performer | Keep, create similar |
| **Good** | Performs well | Keep |
| **Low** | Underperforming | Consider replacing |
| **Learning** | Not enough data | Wait |
| **Pending** | Under review | Wait |

### When to Replace Assets

- After 30+ days of "Low" rating
- When introducing new messaging
- Seasonally (update imagery)
- After significant product changes

## Common Rejection Reasons

**Images:**
- Text overlay (including logos with text)
- Poor image quality
- Misleading content
- Adult content
- Copyrighted material

**Text:**
- Excessive capitalization
- Excessive punctuation (!!!)
- Misleading claims
- Trademarked terms (without permission)
- Phone numbers in text

**Videos:**
- Under 10 seconds
- Copyrighted music
- Misleading content
- Poor audio/video quality
