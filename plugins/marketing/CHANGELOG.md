# Changelog

All notable changes to the Marketing plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-21

### Added

- Initial release of Marketing domain plugin
- Combined Ads, Gen, and Copy tools into single domain plugin
- Unified MCP configuration with google-ads and nano-banana servers
- Domain-level documentation with workflows and use cases
- Project-level installation guidance for token management

### Tools Included

- **Ads** v4.1.2: Google Ads management with subagent architecture
- **Gen** v2.4.1: AI image generation with Gemini
- **Copy** v2.1.1: Copywriting frameworks (Schwartz, Hemingway, Halbert)
