# Content Plugin

Open-source AI plugin that replaces **Writer** and **Editor** roles.

> **Status**: Coming Soon

## Overview

The Content plugin will provide AI-powered long-form writing, editing, and content management. Create, refine, and manage written content at scale.

## Planned Tools

| Tool | Function | Status |
|------|----------|--------|
| **Writing** | Long-form content creation | Planned |
| **Editor** | AI-powered editing and refinement | Planned |
| **SEO Writer** | Search-optimized content | Planned |

## Planned Capabilities

### Long-Form Writing

- Blog posts and articles
- Documentation
- Technical writing
- Book chapters
- White papers

### Editing

- Grammar and style correction
- Tone consistency
- Readability optimization
- Fact checking

### Content Management

- Content calendars
- Version control
- Multi-format export
- Publishing workflows

## Use Cases

- Blog content creation
- Technical documentation
- Marketing content
- Educational materials
- Newsletter writing
- Product descriptions

## Comparison with Copy Plugin

| Aspect | Copy (Marketing) | Content |
|--------|------------------|---------|
| **Focus** | Persuasion, conversion | Information, engagement |
| **Length** | Short-form (headlines, ads) | Long-form (articles, docs) |
| **Goal** | Drive action | Inform and educate |
| **Domain** | Marketing | Content |

The **Copy** tool in the Marketing plugin focuses on direct-response copywriting for ads, emails, and landing pages. The **Content** plugin will focus on longer-form editorial and informational content.
