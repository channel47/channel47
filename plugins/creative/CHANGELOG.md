# Changelog

All notable changes to the Creative plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-01-21

- Initial release of Creative domain plugin
- Integrated Gen tool for AI image generation
- Domain-level documentation with creative workflows
- Style framework references (photography, illustration, 3D, painting, sketch)

### Tools Included

- **Gen** v2.4.1: AI image generation with Gemini (Nano Banana)

### Planned
