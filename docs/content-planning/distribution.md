# Distribution Strategy

Platform-specific tactics, atomization rules, and engagement strategies.

---

## Platform Playbooks

### X (Twitter)

**Role:** Daily touchpoints, build in public, community building

**What works:**
- Threads (5-12 tweets) for deep content
- Standalone observations and insights
- Behind-the-scenes screenshots
- Real numbers and results
- Replies and engagement on related topics

**What doesn't:**
- Pure self-promotion
- Vague motivational content
- Links in first tweet (kills reach)
- Threads without a hook

**Optimal posting:**
- Morning: 8-10am (captures morning scroll)
- Midday: 12-2pm (lunch break traffic)
- Avoid weekends for professional content

**Thread structure:**
```
Tweet 1: Hook (curiosity, bold claim, or question)
Tweet 2: Context/setup
Tweets 3-N: Main content (one idea per tweet)
Final: Summary + CTA (follow, save, reply)
```

**CTA options:**
- "Follow for more [topic]"
- "Save this thread"
- "Which of these resonates most?"
- "Reply with your experience"

### Substack

**Role:** Anchor content, email list building, long-form authority

**What works:**
- In-depth tutorials and guides
- Personal stories with lessons
- Original frameworks
- Data and case studies
- Consistent publishing schedule

**What doesn't:**
- Clickbait without substance
- Irregular posting
- No clear takeaway
- Walls of text (use formatting)

**Structure for articles:**
```
Headline: [Curiosity or Benefit]

Opening hook (1-2 sentences)

The problem/context (2-3 paragraphs)

The solution/insight (bulk of article)
- Use headers to break up sections
- Include examples and specifics
- Add visuals where helpful

Key takeaways (bullet points)

CTA (comment, share, or check out Channel 47)
```

**Subject line formulas:**
- "How I [achieved result] without [common assumption]"
- "The [number] [things] that [benefit]"
- "[Counterintuitive take] about [topic]"
- "Why [common belief] is wrong"

## Engagement Tactics

### First Hour Rule
The first hour after posting is critical for algorithm reach. Plan to:
- Respond to every comment
- Add additional context in replies
- Thank people for sharing/saving

### Comment Strategy
Leave thoughtful comments on:
- Related creators' posts
- Industry news/discussions
- Questions you can answer

**Good comment:**
"I've seen this exact pattern. We switched to [approach] and saw [specific result]. The key was [insight]."

### Building Relationships

3. Add value in replies
4. Over time, they'll reciprocate

- Update any outdated info
- Tweak the hook/angle slightly
- Note that it's an update if relevant
