## Publishing Checklist

### Before Publishing (Substack)
- [ ] Compelling headline (curiosity or benefit)
- [ ] Strong opening line (hook in first sentence)
- [ ] Clear CTA (what should reader do next?)
- [ ] Preview text written
- [ ] Category/tags assigned

### Before Publishing (X)
- [ ] First tweet hooks immediately
- [ ] Thread is scannable (short paragraphs)
- [ ] Includes specific numbers/results where possible
- [ ] Clear CTA in final tweet
- [ ] Scheduled for optimal time (8-10am or 12-2pm)

### Before Publishing (LinkedIn)
- [ ] Professional framing
- [ ] Relatable opening line
- [ ] Uses line breaks for readability
- [ ] Ends with question or CTA
- [ ] Scheduled for weekday (Tue-Thu best)

### After Publishing
- [ ] Respond to early comments (first hour critical)
- [ ] Share in relevant communities (if genuinely helpful)
- [ ] Log performance in notes
- [ ] Extract learnings for next piece
